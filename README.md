# 🔐 Simulador de Ataques de Força Bruta e Dashboard de Cibersegurança

## 📌 Sobre o Projeto

Este projeto tem como objetivo simular ataques de **força bruta (brute force)** e **ataque de dicionário** em um ambiente controlado, com foco educacional em **cibersegurança** e **análise de dados**.

A partir dessas simulações, foi desenvolvido um **dashboard analítico** que demonstra o impacto da complexidade de senhas e a eficiência de diferentes métodos de ataque.

---

## 🎯 Objetivos

* Demonstrar como ataques de força bruta funcionam
* Comparar métodos de ataque (Brute Force vs Dicionário)
* Evidenciar a importância de senhas fortes
* Aplicar análise de dados em cenários de segurança
* Criar visualizações para tomada de decisão

---

## 🧠 Conceitos Abordados

* Ataques de força bruta
* Ataques de dicionário
* Complexidade de senhas
* Tempo de execução de ataques
* Segurança da informação
* Análise exploratória de dados

## 🚀 Como Executar o Projeto

### ▶️ 1. Acesse o Google Colab

Abra o arquivo `.ipynb` no Google Colab.

### ▶️ 2. Execute o código

O simulador irá:

* Gerar múltiplos cenários de ataque
* Testar diferentes senhas
* Coletar métricas (tempo e tentativas)

### ▶️ 3. Criação de tabela

Os resultados serão disponibilizados em forma de tabela.

## 📊 Dashboard de Cibersegurança

O dashboard apresenta:

### 📌 Principais métricas:

* Total de tempo das tentativas
* Total de tentativas

### 📈 Visualizações:

* Tentativas de quebra por tipo de senha
* Análise de complexidade (Fraca, Média, Forte)

---

## 🔐 Insights Gerados

* Senhas fracas são quebradas rapidamente
* Ataques de dicionário são eficientes contra senhas comuns
* O tempo de ataque cresce exponencialmente com a complexidade
* Pequenas mudanças na senha aumentam significativamente a segurança
